# Random-User-API-Performance-Test

## Technology and Tools used
 - JMeter
 
## How to run this project
 - Clone this porject
 - Open jmx file using JMeter
 - Configure it 
 - Run JMeter

   
 ## Prerequisite
  - JAVA 8 or 10 must be installed
  
 ## Testing URL
  - https://random-data-api.com/api/v2/users
